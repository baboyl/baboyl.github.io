Software I use

I really like FOSS (Free Open Source Software) because it respects my privacy and freedom. If you haven't used FOSS then you should really check it out sometime, they are mostly on sites like Github or Gitlab.

Operating System

The OS I run on my computer is Arch Linux, a free lightweight and bleeding edge Linux distrubution. I liked the idea of having a distro which I can make into my own without dealing with a complex install process like with Gentoo.

Web Browser

My main web browser is Ungoogled-Chromium, it's just Chromium without Google's tracking and spying.

File Manager

Pcmanfm is a perfect file manager, it's very light and simple and it looks good. Another nice thing is that is has thumbnail previews which is pretty handy when you store files with random names.

WM/DE

My window manager is dwm, it is a very light and customizable window manager, the tags are very useful for keeping your windows organized.

Media Player

mpv, a very minimal and lightweight media player with plenty of shortcuts.

Text Editor

For basic editing I use Vim, it is very light, runs in a terminal and the keybinds are awesome.

Terminal Emulator & Shell

The terminal emulator I prefer is st (Simple terminal), it is very lightweight and easy to configure if you know C. My default shell is ZSH, it is very good if you would like to have some customizability while also having a bash like experience.

Video/Image editor

For video editing I use Kdenlive, it's pretty good for editing but it did take me a while to get used to how it works. For image editing I use Gimp, it's very simple to use and can edit lots of image types.
